Cyber Security Glossary

Essential cybersecurity terms every student should know

Phishing
A cyber attack that uses disguised emails or messages to trick victims into revealing sensitive information like passwords or credit card numbers.
Malware
Malicious software designed to damage, disrupt, or gain unauthorized access to computer systems. Includes viruses, worms, and ransomware.
Ransomware
A type of malware that encrypts files on a victim's computer and demands payment (ransom) to restore access.
VPN (Virtual Private Network)
A service that encrypts your internet connection and hides your IP address, protecting your data on public Wi-Fi networks.
Two-Factor Authentication (2FA)
An extra layer of security requiring two forms of verification to access an account (password + SMS code or app notification).
Encryption
The process of converting information into a code to prevent unauthorized access. Only someone with the key can decrypt it.
Social Engineering
Psychological manipulation of people into performing actions or divulging confidential information (e.g., impersonation, pretexting).
Firewall
A network security system that monitors and controls incoming and outgoing network traffic based on predetermined security rules.
Zero-Day Exploit
A cyber attack that occurs on the same day a software vulnerability is discovered, before the developer can create a patch.
Brute Force Attack
A trial-and-error method used to decode encrypted data such as passwords by systematically trying all possible combinations.
Man-in-the-Middle (MitM)
An attack where a hacker secretly intercepts and relays messages between two parties who believe they are communicating directly.
DDoS Attack
Distributed Denial of Service - Overwhelming a server with traffic from multiple sources to make it unavailable to legitimate users.
Spyware
Software that secretly monitors user activity and collects personal information without the user's knowledge.
Digital Footprint
The trail of data you leave behind when using the internet, including social media posts, online purchases, and search history.
Patch
A software update that fixes security vulnerabilities or improves performance. Regular patching is essential for security.
Social Media Impersonation
Creating fake social media profiles pretending to be someone else, often used for scams, harassment, or reputation damage.
